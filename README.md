## Install

<p align="left">
  <ul>
    <li>Go to the Kodi file manager.</li>
    <li>Click on "Add source"</li>
    <li>Go to "Addons"</li>
    <li>In Addons, install an addon from zip.  When it asks for the location, select "BotAllen Repository", and install <a href="repository.sandeep1027-2.0.0.zip">repository.botallen-2.0.0.zip</a>.</li>
    <li>Go back to Addons install, but this time, select "Install from repository"</li>
    <li>Select the "BotAllen Repository"</li>
    <li>Go into the "Video add-ons" section in the repo, and you'll find all video add-ons</li>
  </ul>
</p>
